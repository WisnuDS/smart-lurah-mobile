# SmartLurah

## Project Kuliah Pemrograman Berbasis Mobile

Smart Lurah adalah aplikasi yang dapat memudahkan masyarakat kelurahan dalam mengurus berkas berkas kelurahan, seperti surat pengantar ktp, surat pindah, dll.
